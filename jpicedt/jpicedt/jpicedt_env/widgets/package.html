Des componsants Swing utilisant dans les classe de l'application JPicEdt (jpicedt.JPicEdt) et de son installeur (installer.Install). Tous ces composants peuvent toutefois être utilisé indépendamment.

Ces classes étaient à l'origines dans le paquetage jpicedt.widget, mais ont été déplacée vers jpicedt.jpicedt_env.widget pour pouvoir être utilisée par l'installeur.
